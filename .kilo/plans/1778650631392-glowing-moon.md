# Profile Settings Page UI Fixes

## Issues
1. **Title-description gap**: The gap between the "Account Settings" title and description is too large.
2. **Stretched form inputs**: Form inputs in ProfileForm and PasswordChangeForm stretch too wide.
3. **Responsiveness**: Inputs need responsive centering across breakpoints.

---

## Change 1: ProfileSettingsPage.tsx

**File**: `frontend/src/features/users/pages/ProfileSettingsPage.tsx`

**Lines 47-66** — Tighten the gap between title and description:

```tsx
// BEFORE (line 48-54):
<Typography
  variant="h4"
  sx={{
    color: "primary.main",
    fontWeight: 800,
    letterSpacing: "-0.5px",
    fontSize: { xs: "1.5rem", sm: "1.75rem", md: "2.05rem" },
    mb: 0.5,
  }}
>
  Account Settings
</Typography>
<Typography
  variant="body1"
  color="text.secondary"
  sx={{ fontSize: "1.1rem", maxWidth: 600 }}
>

// AFTER:
<Typography
  variant="h4"
  sx={{
    color: "primary.main",
    fontWeight: 800,
    letterSpacing: "-0.5px",
    fontSize: { xs: "1.5rem", sm: "1.75rem", md: "2.05rem" },
    mb: 0.25,
  }}
>
  Account Settings
</Typography>
<Typography
  variant="body1"
  color="text.secondary"
  sx={{ fontSize: "1.1rem", maxWidth: 600, mt: 0.5 }}
>
```

Key changes: `mb: 0.5` → `mb: 0.25` on title, added `mt: 0.5` on description.

## Change 2: ProfileForm.tsx

**File**: `frontend/src/features/users/components/ProfileForm.tsx`

**Lines 102-106** — Center the form grid and constrain its width:

```tsx
// BEFORE:
<Box
  component="form"
  onSubmit={handleSubmit(onFormSubmit)}
  sx={{ display: "grid", gap: 2.5 }}
>

// AFTER:
<Box
  component="form"
  onSubmit={handleSubmit(onFormSubmit)}
  sx={{
    display: "grid",
    gap: 2.5,
    maxWidth: { xs: "100%", sm: 480 },
    mx: "auto",
  }}
>
```

This prevents the form from stretching full-width on large screens and centers it within the card with a max-width constraint.

## Summary

| File | What changed |
|---|---|
| `ProfileSettingsPage.tsx` | Reduced title bottom margin (`0.5` → `0.25`), added `mt: 0.5` to description |
